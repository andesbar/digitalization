# DIGITALIZATION Secured Digital Ecosystem
**National Core Architecture Indonesia Goverment**

A highly secure, multi-level government web architecture spanning from Village (Desa) to Provincial levels.

### Tech Stack:
- **Core Engine:** PHP 8.x (Custom MVC)
- **Database:** PostgreSQL 18
- **Security:** Anti-Source Code Inspection, Multi-Schema Multitenancy
- **Architecture:** Distributed Core System

0. INFRASTRUKTUR & DEVOPS (LAPISAN KEAMANAN TERLUAR SERVER)

│  ├─ waf.conf                 ← Web Application Firewall: Aturan blokir bot jahat & payload SQLi/XSS.
│  ├─ ddos_protect.conf        ← Rate limiting di level server (membatasi jumlah koneksi per detik per IP).
│  └─ ssl_tls13.conf           ← Memaksa penggunaan HTTPS (TLS 1.3) dan cipher suite kriptografi modern.
├─ postgresql/

└─ ci-cd/
   └─ security_scan.yml        ← Pipeline analisis kode statis (SAST) sebelum rilis ke server.
